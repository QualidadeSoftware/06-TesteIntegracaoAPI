# 📁 collections/

Aqui ficam as coleções exportadas do Postman/Bruno.

## O que tem aqui

- `EXEMPLO-atividade-teste-api.postman_collection.json` — **referência** com 9 requests prontos (6 positivos + 3 negativos). **Não é a sua entrega**.

## O que você precisa adicionar

Sua coleção exportada, com nome no padrão:

```
atividade-teste-api.postman_collection.json
```

## Como exportar do Postman

1. Clique nos três pontinhos (⋯) ao lado do nome da sua coleção
2. Selecione **Export**
3. Escolha **Collection v2.1** (recomendado)
4. Salve o `.json` nesta pasta

## Como exportar do Bruno

1. Clique com o botão direito na coleção
2. **Export As → Postman Collection**
3. Salve o `.json` nesta pasta

## Verificando a entrega

Após exportar, abra outro Postman (ou peça a um colega) e tente **importar o seu arquivo**. Se ele funciona em outra máquina, está pronto para entrega.
